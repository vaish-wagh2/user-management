# React CRUD User Management App (React 19 + TypeScript)

A simple and extensible React 19 + TypeScript CRUD application for managing users.  
This project demonstrates clean architecture, API integration, form validation, and a configuration-driven dynamic form approach.

---

##  Live Demo

- **Live URL:** https://user-management-2vng8evu9-vaishs-projects-705e8e61.vercel.app/


##  Features

###  User Fields
- First Name  *(required,validation)* 
- Last Name *(required)*  
- Phone Number *(required, validated)*  
- Email Address *(required, validated)*  

---

###  CRUD Operations
- Create User  
-  View / List Users  
-  Edit User  
-  Delete User  

---

###  Technical Highlights
- React 
- TypeScript
- Configuration-driven dynamic form
- Bootstrap 5 UI
- API integration using Axios
- Mock api
- Easily replaceable with a real backend
- Clean and modular code structure
